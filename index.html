Mengenal Semantic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada diwebsite kita, tag artikel bisa kita gunakan dengan element header dan footer. Karena tag article memiliki type/sifat block, maka lebarnya sesuai page jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai pagi jika kita tidak menentukan lebarnya.
